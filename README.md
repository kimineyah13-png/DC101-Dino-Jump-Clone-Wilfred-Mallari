# Rock Paper Scissors Game

Rock Paper Scissors is a simple browser-based game where the player competes against the computer.  
The player selects rock, paper, or scissors, and the computer makes a random choice.  
Scores are tracked, and the first to reach 5 points wins the game.

---

## Game Features
- Player vs computer gameplay
- Score tracking for both player and computer
- Displays player and computer choices
- Automatic winner announcement at 5 points
- Reset button to restart the game

---

## Technologies Used
- HTML5 – Structure of the game
- CSS3 – Styling and layout
- JavaScript (Vanilla JS) – Game logic and interactivity
